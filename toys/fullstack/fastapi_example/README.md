## Description

Creates a sample FastAPI app.

## Quickstart

```shell
docker compose up
```

The base url is <http://localhost:4041> and the endpoints are:

- <http://localhost:4041/name> (Random name.)
- <http://localhost:4041/text> (Random text.)
- <http://localhost:4041/int> (Random int.)
