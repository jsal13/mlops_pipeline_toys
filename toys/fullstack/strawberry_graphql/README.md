# Strawberry GraphQL Example

## Last Updated

2023-11-18

## Status

**Working.**

## Description

Example of a schema in GraphQL via Strawberry.

## Quickstart

```shell
docker compose up
```

Then go to <localhost:8000/graphql>.

To get example data back, one possible query is:

```text
{
  users {
    name
    address
  }
}
```
