# Log Generator

## Last Updated

2023-11-18

## Status

**Working.**

## Description

Generates [CLF logs][CLF Logs] at <http://localhost:4041/log> (or you can use `generate_batch_random_log_data` to generate batch log data).

## Quickstart

```shell
docker compose up
```
